## 📌 Overview

Expense Analytics Dashboard is a Python-based analytics project developed to analyze expense records from CSV files and generate meaningful financial insights.

The system processes raw transaction data using Python to calculate key performance indicators (KPIs), category-wise spending, monthly trends, and expense summaries. The processed analytical data is then visualized through an interactive dashboard.

---

## ✨ Key Features

### 🐍 Python Analytics
- Process CSV expense data
- Calculate total expenses
- Generate category-wise analysis
- Perform monthly expense tracking
- Detect highest & lowest spending month
- Calculate average daily expenses
- Generate structured JSON analytics

### 📈 Dashboard Visualization
- KPI summary cards
- Expense insights generation
- Category breakdown table
- Monthly summary cards
- Doughnut chart for category distribution
- Bar chart for expense comparison

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/GurbaniGudnu/Expense-Analytics-Dashboard.git
```

### 2️⃣ Navigate to Project Directory

```bash
cd Expense-Analytics-Dashboard
```

### 3️⃣ Run Python Analytics Script

```bash
python expense_analytics.py
```

This generates:

```text
analytics.json
```

### 4️⃣ Open Dashboard

Open:

```text
index.html
```

in your browser.
